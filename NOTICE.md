- [uv](https://github.com/astral-sh/uv): optional native installer helper,
  dual licensed Apache-2.0 OR MIT. The bootstrap script downloads it at install

  workflow adapter and does not vendor or import WorkBuddy source code.

## Source disclosure

Machine-readable source disclosure lives in:

- `config/accounts.json` for WeChat public accounts.
- `config/external_sources.json` for RSS, Atom, podcast, and pending web
  sources.
- `docs/disclosure.md` for the human-readable disclosure summary.
- `docs/source-candidates.md` for rationale, quality notes, and candidate
  status.
