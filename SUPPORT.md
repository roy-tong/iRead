# Support

Start with the [60-second quick start](README.md#60-秒开始) and [Agent installation guide](docs/agent-installation.md).

- Installation, collection, scheduling, or report failure: use the [bug report form](https://github.com/roy-tong/iRead/issues/new?template=bug-report.yml).
- Need help defining a field or source portfolio: use the [subscription request form](https://github.com/roy-tong/iRead/issues/new?template=research-profile.yml).
- Correct or recommend a source: use the [source contribution form](https://github.com/roy-tong/iRead/issues/new?template=source-contribution.yml).
- Security vulnerability: follow [SECURITY.md](SECURITY.md) instead of opening a public issue.

Never post access tokens, cookies, private session data, WeChat QR codes, authorization files, private local paths, or third-party article bodies.
